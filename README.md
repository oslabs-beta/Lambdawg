## About LAMBDAWG

<p>LAMBDAWG is a free tool designed to simplify the management and visualization of AWS Lambda functions and metrics. It provides easy access and straightforward insights for AWS developers, eliminating the need to navigate complex documentation. With LAMBDAWG, you can quickly visualize the correlation between Lambda functions within your app and keep an organized log of errors, all in one place. This tool aims to speed up your development process and enhance your overall experience with AWS Lambda.</p>

<ol>
<h1>Key Features</h1>
<li>Centralized Error Log Management: Keep an organized list of error logs across your AWS Lambda Functions in a single location for easy access and troubleshooting.</li>
<li>Invocation Monitoring: Gain visibility into the number of invocations for your Lambda functions, allowing you to track usage and performance.</li>
<li>Function Execution Duration: Easily track the duration of each Lambda function's execution to identify potential bottlenecks and areas for improvement.</li>
<li>Response Time Analysis: Measure the time taken to generate the response for each function, helping you evaluate efficiency and optimize user experiences.</li>
</ol>

## Getting Started
<ol>

## Contributing

We welcome contributions to LAMBDAWG from the community. Whether you have a great idea to enhance the tool or you've discovered and fixed a bug, we appreciate your involvement. Follow the steps below to contribute:

<ol>
  <li>Fork the LAMBDAWG repository.</li>
  <li>Clone the repository to your local machine using the command: `<strong>git clone your_repo_url</strong>`</li>
  <li>Create a new branch for your feature: `<strong>git checkout -b your/amazingFeature</strong>`</li>
  <li>Make the necessary changes and commit them with a brief comment: `<strong>git commit -m "Quick comment about your amazing feature"</strong>`</li>
  <li>Push the changes to your branch: `<strong>git push origin your/amazingFeature</strong>`</li>
  <li>Open a pull request to submit your changes for review and integration</li>
</ol>
We value community feedback and participation. Don't forget to give LAMBDAWG a ⭐️ and share it with your friends. <br>
Together, we can make LAMBDAWG even better!
